### Relaciones Explicadas

**User â†’ Exercise (Custom)**
- Un usuario puede crear N ejercicios personalizados
- Los ejercicios globales no tienen `created_by`

**User â†’ Routine**
- Un usuario tiene N rutinas
- Cada rutina es privada al usuario

**Routine â†” Exercise (via RoutineExercise)**
- N:M con metadata (order, target_sets, target_reps)
- RoutineExercise es el template del ejercicio en la rutina

**Workout â†’ Routine (opcional)**
- Un workout puede iniciar desde una routine (copia ejercicios)
- O puede ser freestyle (routine = NULL)

**Workout â†’ WorkoutExercise â†’ Set**
- Un workout tiene N ejercicios realizados
- Cada ejercicio tiene N sets (series)
- Set guarda: peso, reps, completed, RPE
